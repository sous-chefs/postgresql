## Requirements

### Platforms

- Amazon Linux
- Debian 9+
- Ubuntu 18.04+
- Red Hat/CentOS/Scientific 7+

### PostgreSQL version

We follow the currently supported versions listed on <https://www.postgresql.org/support/versioning/>

### Chef

- Chef 18

### Cookbook Dependencies

- `yum` >= 7.2.0 (for `dnf_module` resource)
